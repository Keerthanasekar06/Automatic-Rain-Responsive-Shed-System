# Automatic-Rain-Responsive-Shed-System
   This project is an Arduino-based automation system that detects rainfall using a rain sensor and automatically activates a protective shed mechanism using a servo motor. The system can be used to protect solar panels, outdoor equipment, and other exposed infrastructure from rain damage.
   
Objectives
- Detect rainfall using a rain sensor.
- Automatically control a servo motor.
- Protect outdoor equipment from weather conditions.
- Demonstrate embedded automation concepts.

Components Used
- Arduino Uno
- Rain Sensor Module
- Servo Motor
- Breadboard
- Jumper Wires
- Power Supply

Working Principle
1. The rain sensor continuously monitors rainfall.
2. Arduino reads the sensor output.
3. When rain is detected, the servo motor rotates to close the protective shed.
4. When rain stops, the servo returns to its original position.

Applications
- Solar panel protection systems
- Smart home automation
- Agricultural equipment protection
- Weather-responsive outdoor structures
